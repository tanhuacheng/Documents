大多数特殊字符在中括号表达式内出现时失去它们的意义. 不过也有一些例外:
    如果 ] 字符不是第一项, 它结束列表. 要匹配列表中的 ] 字符, 将它放在第一位, 紧跟在 [ 后面.
    \ 字符继续作为转义符. 若要匹配 \ 字符, 使用 \\


转义:

\b      一个字边界. 它的前一个和后一个字符不全是 \w
\B      非字边界
\cx     x 指示的控制字符. \cM -> 回车; \cL -> 换页, x in [A-Za-z], 否则假定 c 就是 'c' 本身
\d      数字. 等效于 [0-9]
\D      非数字
\f      换页. 等效于 \x0c \cL
\n      换行. 等效于 \x0a \cJ
\r      回车. 等效于 \x0d \cM
\s      空白字符. 等效于 [ \f\n\r\t\v]
\S      非空白字符
\t      制表符. 等效于 \x09 \cI
\v      垂直制表符. 等效于 \x0b \cK
\w      字类字符. 等效于 [A-Za-z0-9_]
\W      非字类
\xn     n 是两个十六进制码, 匹配对应的 ASCII 码
\n      n 是一个正整数, 表示捕获引用, 匹配对应的捕获, 或没有对应的引用时匹配八进制转义码
\un     n 是四位十六进制数表示的 Unicode 字符. 例如 \u00A9


vim:

/\v     使用正则表达式查找
'<' '>' 对应 \b 在单词前后
